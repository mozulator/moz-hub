# Notion-Style UI Design Prompt

A comprehensive design prompt for generating user interfaces inspired by Notion’s clean, minimal, and structured aesthetic.

---

## Design Goal
Create a modern, minimalistic web interface inspired by Notion’s calm and elegant workspace design, focusing on clarity, spacing, and simplicity.

---

## Core Aesthetic
- Ultra-minimal, distraction-free layout  
- Soft grayscale palette  
- Generous spaciousness and comfortable margins  
- Thin, subtle dividers and borders  
- Typography-first design  
- Calm, neutral interaction states  

---

## Color Palette

### Base Colors
- Background: `#FFFFFF` or `#FAFAFA`
- Secondary Background: `#F6F6F6`

### Text
- Primary: `#2E2E2E`
- Secondary: `#6F6F6F`

### Borders / Lines
- `rgba(0,0,0,0.08)`

### Optional Accents
- Muted blue: `#4C7FF0`
- Soft beige: `#F5E6CC`
- Orange folders: `#FF8C00` (for tree view folders)

---

## Typography
- Use clean geometric sans-serif fonts such as Inter, Helvetica Neue, or Söhne  
- Weights: 300, 400, 500  
- Titles slightly larger, rarely bold  
- Line height: 1.5–1.75  
- Wide spacing between sections for readability  

---

## Layout Structure
- Use an 8px / 12px spacing grid  
- Large padding and airy sections  
- Modular block-based layout  
- Minimal shadows (e.g., `0 1px 3px rgba(0,0,0,0.04)`)  

---

## Components

### Buttons
- Border: `1px solid rgba(0,0,0,0.08)`  
- Background: white  
- Icon and text horizontally aligned  
- Hover: `rgba(0,0,0,0.05)`  
- Border-radius: `6px`  
- Optional subtle elevation  

### Inputs
- White background  
- Soft border: `rgba(0,0,0,0.1)`  
- Calm, soft focus ring  
- Generous padding  

### Cards and Panels
- White background  
- Very light or no border  
- Border-radius: `8px`  
- Optional micro-shadow  

### Navigation Sidebar
- Monochrome icons  
- Collapsible sections  
- Subtle hover states  
- Avoid strong accent colors

### Icons
- Use Font Awesome 6.5.1 for all icons
- Icons should be monochrome (grayscale) in most contexts
- Icon size: 14-16px for standard use
- Orange folder icons (`#FF8C00`) for tree view components
- Icons align horizontally with text

### Tree View
- Hierarchical file/folder structure display
- Orange folder icons (`#FF8C00`) for visual distinction
- Collapsible/expandable nodes with chevron indicators
- Smooth expand/collapse animations (200ms)
- Subtle hover states on nodes
- Indented children for visual hierarchy
- File icons remain monochrome (grayscale)  

---

## Interaction Style
- Smooth transitions (150–200ms)  
- Soft hover states  
- No heavy color shifts  
- No loud animations  
- Calm and subtle interactions  

---

## Overall Tone
- Calm  
- Neutral  
- Organized  
- Effortless  
- Easy to scan and interact with
